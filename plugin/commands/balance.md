# RunAPI Balance

Call `mcp__runapi__check_balance`.

If the tool returns an API key error, call or recommend the RunAPI `login` tool for browser login. For headless hosts, explain `RUNAPI_API_KEY` or `~/.config/runapi/config.json`.
Do not ask for the user's API key in chat.
Do not print raw JSON unless the user asks.
